# Depreciation System

A comprehensive depreciation management system built with PHP backend and modern frontend styling.

## Overview

This project implements a depreciation system for tracking and managing asset depreciation. It provides a complete solution for calculating, monitoring, and reporting on asset depreciation over time.

## Technology Stack

- **Backend:** PHP (15.8%)
- **Styling:** CSS (30.5%), SCSS (26.1%), Less (25.8%)
- **Frontend:** JavaScript

## Features

- Asset depreciation calculations
- Depreciation tracking and reporting
- Asset management interface
- Modern responsive design

## Project Structure

```
Depreciation-System/
├── assets/          # CSS, SCSS, Less stylesheets
├── src/             # PHP backend code
├── js/              # JavaScript files
└── README.md        # This file
```

## Installation

1. Clone the repository:
```bash
git clone https://github.com/theinvinciblebe/Depreciation-System.git
cd Depreciation-System
```

2. Install dependencies (if using Composer):
```bash
composer install
```

3. Configure your web server to point to the project directory
